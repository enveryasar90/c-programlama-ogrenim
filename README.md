# C Programlama Öğrenim Süreci

Bu repo, C programlama dilini sıfırdan öğrenirken yazdığım örnek kodları ve alıştırmaları içeriyor. Konular, temel seviyeden başlayarak sırasıyla ilerleyecek şekilde düzenlendi.

## Kapsanan Konular

| # | Konu | Açıklama |
|---|------|----------|
| 1 | Temel Yapı ve Çıktı | `main()`, `printf`, derleme süreci |
| 2 | Değişkenler ve Veri Tipleri | `int`, `float`, `char`, format belirteçleri |
| 3 | Operatörler | Aritmetik işlemler, int/float bölme farkı |
| 4 | Karar Yapıları | `if`, `else if`, `else` |
| 5 | Döngüler | `for`, `while` |
| 6 | Diziler | Tek boyutlu diziler, döngü ile gezme |
| 7 | Fonksiyonlar | Parametre, `return`, fonksiyon tanımlama |
| 8 | String'ler | `string.h` (`strlen`, `strcat`, `strcpy`, `strcmp`) |
| 9 | Pointer'lar | `&`, `*`, referans ile fonksiyon çağırma |
| 10 | Struct | Yapılar, birden fazla veri tipini gruplama |
| 11 | Dinamik Bellek Yönetimi | `malloc`, `free` |
| 12 | Dosya İşlemleri | `fopen`, `fprintf`, `fgets`, `fclose` |

## Kullanılan Araçlar

- **Derleyici:** GCC
- **Editör:** VS Code (Code Runner eklentisi ile)

## Notlar

Her dosya, ilgili konunun alıştırma çözümünü içerir. Bazı dosyalarda ek denemeler veya kendi kurguladığım ek problemler de yer alıyor.
